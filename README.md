## 下一代图书计划
这个书单希望收集一系列质量优秀的书籍。筛选的标准是：
1. 自己完整阅读过，认为确实好。
2. 认为自己的下一代值得一读的书。（虽然还没有女朋友，但是如果这么思考的话，应该有助于选出经得起时间考验的书籍）
